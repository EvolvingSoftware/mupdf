# Docrafta Integration with MuPDF

This repository contains the MuPDF library customized for the Docrafta project.

## Project Overview

Docrafta is a document processing platform that leverages the powerful MuPDF library for PDF manipulation and rendering.

## Customizations

- Added Docrafta-specific APIs for document processing
- Integrated with Docrafta's authentication system
- Enhanced performance for large document processing
- Added support for Docrafta's custom document formats

## Build Instructions

```bash
make tools
```

## Usage

The customized MuPDF library can be used through the standard mutool interface with additional Docrafta-specific commands.
